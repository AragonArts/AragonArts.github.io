Aragon's Backdoor Scanner (ABS) is a program that reads all files in a specified directory and searches using filters for triggers that the user might be interested in, useful for finding backdoors.

using its new modern lite UI, the program is lightweight and does not lag the system during the search.

the search doesn't take longer than 3 seconds to scan through all of the files in the directory.
